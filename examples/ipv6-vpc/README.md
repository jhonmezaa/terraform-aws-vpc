# IPv6 VPC Example

Dual-stack VPC with full IPv6 support.

## Features
- IPv6 enabled VPC
- Egress-only Internet Gateway (IPv6 outbound only)
- Auto-assign IPv6 addresses to instances
- Dual-stack (IPv4 + IPv6)

## Cost: ~$96/month
- NAT Gateways: ~$96/month (for IPv4)
- IPv6: Free

## Use Cases
- Modern applications requiring IPv6
- Compliance requirements for IPv6
- Future-proof infrastructure
